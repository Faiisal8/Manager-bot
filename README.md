# Manager-bot





## Before starting install requirements

1. npm init -y 
2. npm i
3. npm i discord.js
4. npm i ascii-table
5. npm i dayjs
6. npm i discord-modals
7. npm i moment
8. npm i mongoose
9. npm i nodemon
